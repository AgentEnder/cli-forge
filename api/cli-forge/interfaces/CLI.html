Interface: CLI<TArgs>

The interface for a CLI application or subcommands.

cli is provided as a small helper function to create a new CLI instance.

Example​

import { cli } from 'cli-forge';

cli('basic-cli').command('hello', {
  builder: (args) =>
   args.option('name', {
     type: 'string',
   }),
  handler: (args) => {
    console.log(`Hello, ${args.name}!`);
  }).forge();

errorHandler(handler): CLI<TArgs>

Registers a custom global error handler for the CLI. This handler will be called when an error is thrown during the execution of the CLI and not otherwise handled. Error handlers should re-throw the error if they cannot handle it, s.t. the next error handler can attempt to handle it.

Typically called with an Error object, but you should be prepared to handle any type of error.
